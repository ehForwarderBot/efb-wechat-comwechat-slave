部分已经支持的命令：

```
/search - 按关键字匹配好友昵称搜索联系人

/addtogroup - 按wxid添加好友到群组

/getmemberlist - 查看群组用户wxid

/at - 后面跟wxid，多个用英文,隔开。

/changename - 修改群组名称

/getstaticinfo - 可获取friends, groups, contacts信息
```
